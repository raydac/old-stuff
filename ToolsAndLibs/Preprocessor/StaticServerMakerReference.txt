Описание системы Static Server Maker (SSM)

Версия: 1.00
Дата:   13 июня 2005

Автор: И.А.Мазница




Введение
-----------

Система SSM предназначена для формирования статического веб контента на
базе XML файлов. Система формирует HTML странички и графические изображения на базе заполнения шаблонов из XML файлов
и построения иерархической структуры, базируясь на конфигурациях, прописанных в XML файлах.

Список конфигурационных файлов
---------------------------------

server.xml - файл описывает иерархию сервера и языковые настройки, фактически содержит метаданные по серверу


Описание формата файла "server.xml"

<server name="server_name" indextemplate="indexfile_template">
    <languages>
        <language id="rus" txt="RUS"/>
        <language id="eng" txt="ENG"/>
    </languages>

    <hierarhy>
        <item id="NEWS">
            <str langid="rus" value="Новости"/> - написание на языке
            <str langid="eng" value="News"/> - написание на языке
            <datafile file="file_name"/> - файл, содержащий информацию для данного раздела
        </item>
    </hierarhy>
</server>
