				TORWART

BACKGROUND : цвет заднего фона по умолчанию 0A0063
IMAGES : относительный путь к картинкам
ORIENTATION : ориентация игрового поля ( V - вертикальная, H - горизонтальная) по умолчанию горизонтальная
FIELDWIDTH: ширина игрового поля (10<width<44, по умолчанию 40)
FIELDHEIGHT: высота игрового поля (10<height<44, по умолчанию 40)
BALLS: количество мячей (до 1000, по умолчанию 100)
