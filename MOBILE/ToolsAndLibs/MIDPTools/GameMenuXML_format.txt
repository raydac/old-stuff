Game menu XML формат

<GameMenu>
	<item id="itemid" text="textid" flags="NONE|ONENTER|ONEXIT|OPTIONAL|CUSTOM" type="BUTTON|BUTTONLIST|RADIOLIST|CHECKLIST|TEXT"
		<subitem id="id" text="textid" flags="NONE|ONSTATE|ONENABLE"/>
	</item>
</GameMenu>

Тег "item"
-------------------
Описывает один элемент меню
Флаги
ONENTER - отработать функцию при выборе пункта.
ONEXIT - отработка при переходе пункта в неактивное состояние.
OPTIONAL - показывает что на добавление пункта в меню, требуется подтверждение.
CUSTOM - показывает что пункты меню формируются извне.
NONE - нет флагов (по умолчанию).

Типы
BUTTON - кнопка, при нажатии вызывается отработчик (по умолчанию)
BUTTONLIST - пункты меню выводятся как кнопки нажатие на которые отрабатывается.
RADIOLIST - пункты меню выводятся как радиокнопки.
CHECKLIST - пункты меню выводятся как чекбоксы.
TEXT - пункты меню содержат текст, который следует отобразить.

Тег "subitem"
-------------------
Флаги
ONSTATE - вызывает отработчик при смене состояния пункта.
ONENABLE - вызывает запрос на разрешение вывода пункта меню.
NONE - нет флагов (по умолчанию).


Бинарный формат файла, описывающего игровое меню

ID    (0x45328300) Integer  (заголовок)
-----------------------------------------
DATA SIZE Unsigned short (размер блока данных, идущего за данным полем)
-----------------------------------------
VERSION (0x0100) Unsigned short (версия)
-----------------------------------------
ITEMS NUMBER  Unsigned byte (Количество пунктов меню)
-----------------------------------------
ITEM_0
...     Данные пунктов меню
ITEM_1
-----------------------------------------

Формат хранения пункта меню
-----------------------------------------
ItemID Unsigned byte (идентификатор пункта)
TextID Unsigned byte (Текст пункта)
Type  Unsigned byte (Тип пункта меню)
    0 - BUTTON
    1 - BUTTONLIST
    2 - RADIOLIST
    3 - CHECKLIST
    4 - TEXT

Flags Unsigned byte (Флаги)
    0 - NONE
    1 - ONENTER
    2 - ONEXIT
    4 - OPTIONAL
    8 - CUSTOM

Subitems number Unsigned byte (Количество подпунктов)

Subitem_0
...
Subitem_1

Формат хранения подпунктов меню
--------------------------------------------
SubItemID unsigned byte (Идентификатор подпункта)
TextID unsigned byte (Идентификатор текста связанного с подпунктом)
Flags  unsigned byte (Флаги)
    0 - NONE
    1 - ONENABLE
    2 - ONSTATE
